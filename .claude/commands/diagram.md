Follow this workflow:
1. Read relevant source files to identify components, services, databases, and connections
2. Plan the diagram layout — choose tiers, group related components, map arrow connections
3. Create the diagram using `create_view` with all elements at once
4. Export as PNG via `export_scene format="png"` and save the file
5. Save the `.excalidraw` source file for future editing
